### pywinauto介绍
pywinauto是一个用于自动化Python 模块，适合Windows系统的软件（GUI），可以通过Pywinauto遍历窗口（对话框）和窗口里的控件，也可以控制鼠标和键盘输入
官网文档：https://pywinauto.readthedocs.io/en/latest/
